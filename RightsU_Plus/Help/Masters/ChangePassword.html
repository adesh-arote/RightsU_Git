Change Password

If user wants to change his/her password, they can go to Change Password screen and update the existing password.

For that, enter old password. Enter New password and Confirm new password.

Note:

- If password is updated, user won't be able to use old password to log into system.

- Password should be more than 8 characters and combination of numbers and letters (alphanumeric).

- New password and Confirm Password must match.
