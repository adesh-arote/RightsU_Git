Music Album

This module will be accessible only to the User who has rights for this module. The User who has rights for individual action (add/modify/activate/deactivate/view) can add, modify, view, activate, deactivate and change the Music Album Name in the system.

This module is used to add a Music Album in the system which will be used later to assign the Music Album.

Clicking on Music Track>>Music Album opens the Music Album List page with all added Music Albums (as below).

User can also search for the Music Album by clicking on the Search Button on the right side of the screen.

User can search through Music Album and Talent in the Search box.

Add Music Album:

To add a Music Album click on 'ADD' button on the Music Album List page, which will open a blank tab (like below) to add a new Music Album.

Add Music Album will have the following fields:

Music Album Name: The user can add the name of the Music Album.

Album Type: The user can select the type of the Album, whether it is a Movie Album or a Music Album.

Talent: The user can add the Talent for the Music Album. The Talent data is populated from Masters>>Commercial>>Talent List. The user can select from a list of singers added earlier. More than one talent can be selected at a time.

Status: This shows the status of the Music Album whether it is Active or Inactive. The Status cannot be changed.

Action: The user can 'Save' or 'Cancel' saving the Music Album.

SAVE: If the user clicks on the 'Save' button, it will save the Music Album and will show on the Music Album List page.

CANCEL: Clicking on 'Cancel' will cancel the Edit operation and will go back to the previous entry on the Music Album List page without any changes.

Edit Music Album Name:

The 'Edit' button is located on the Music Album List page in the Action column, and is indicated by a Pencil icon. Clicking on the 'Edit' button, opens 'Edit Music Album' tab (like below) to edit an existing Music Album.

Edit Music Album will have the following fields:

Music Album Name: The user can edit the existing name of the Music Album.

Status: This shows the status of the existing Music Album whether it is Active or Inactive.

Action: The user can 'Save' or 'Cancel' saving the edited Album.

SAVE: If the user clicks on the 'Save' button, it will save the Music Album and will show on the Music Album List page.

CANCEL: Clicking on 'Cancel' will cancel the Edit operation and will go back to the previous entry on the Music Album List page without any changes.

Activate or Deactivate a Music Album:

Activate Music Album: 'Thumbs Up' icon activates that particular Music Album

Deactivate Music Album: 'Thumbs Down' icon deactivates that particular Music Album.
