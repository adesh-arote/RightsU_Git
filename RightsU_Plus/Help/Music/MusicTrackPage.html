Music Track List

This module will be accessible only to the user who have rights for this module. User can add, modify, view, activate, deactivate and Change Music Label in the system and it will be done by the user who has rights for individual action (add/modify/activate/deactivate/view).

This module is used to add music track in the system which will be used later to assign the Music Track.

CLick on Music Track - Music Track List which opens the Music Track List page with all added titles as below.

User can also search for the Music Track by clicking on the Advance Search Icon on the right side of the screen.

Using Advance Search, user can search through Music Track, Music Language, Music Label, Year of Release, Singer, Song Star Cast, Music Composer, Lyricist, Genres, Music Theme, Music Tag in Advance Search.

Add Music Track

To add a music track click on 'Add Music Track' button on the Music Track List page. Clicking on the 'Add Music Track' button opens 'Add Music Track' page as shown in screenshot below

Add Music Track will have following fields :

Music Track: The user needs to add the name of the Music Track. This is a mandatory field

Movie/ Album: The user can select the movie or album from the dropdown list or create a new Movie or Album using the add button. This is a mandatory field.

Music Language: User needs to select the language for the Music Track. Music Language data is populated from Music Language Master. This is a mandatory field.

Music Label: User have to select the Music Label from the dropdown list. This is a mandatory field

After clicking on SAVE button, it will save the Music Track and will show on the Music Track List page. Clicking on ADD ADDITIONAL METADATA will redirect user to the page below:

Here user has the option to Upload the Music Track image along with following fields.

Singer: This shows list of singers added earlier. User can select more than one at a time.

Lyricist: This shows list of lyricist added earlier. User can select more than one at a time.

Music Type: The user can select the type of the Music from the dropdown list.

Song Start Cast: This shows list of song start cast added earlier. User can select more than one at a time.

Length (mins): This is to enter the length of the Music Track in minutes.

Year of Release: This is to enter year of release of the Music Track.

Music Composer: This shows the list of Music Composer added earlier. User can select more than one at a time.

Genre: Shows list of genres added earlier in Music Genre Master. User can select more than one.

Music Theme: This shows the list of Music Theme added earlier. User can select more than one at a time.

Tag: This is to enter the Tag related to the music track.

SAVE: Clicking on save will save the Music Track in the system.

CANCEL: Clicking on Cancel is to cancel the add operations and goes back to previous window.

Add Singer

This facility is to add Singer into system directly from Music Track master without going to Talent master so that the added entry will display in Singer drop down to select.

Click on "ADD" to display the Add Talent window page as below.

Talent:Enter Talent name in the Text box.

Role: By default Singer will be selected and multiple check box can be selected such as Star Cast and Producer.

Gender: Can select either Male or Female or NA through radio button.

SAVE: Click Save button to save data to the database, an alert message appears as “Talent added successfully”, click “OK” a new Talent added into the Talent master module as well as display in the current process.

CANCEL: Click Cancel to cancel the add operations and previous window disappear and data should not saved.

Follow same process to add new Lyricist, Song Star Cast, Music Composer in the system as shown for Singer.
