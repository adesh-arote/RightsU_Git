Music Label

Music Label comes under Music Track Module

Click Add Button on top right corner of the page as shown in the screenshot to add a new Label

User will enter Label name in text box

Click on Save button user will get alert message “Music Label Added successfully”./p>

Click on Cancel Button, the Record will not get updated.

Note: Label Name will be Unique in the system.

By default Music Label status will show ACTIVE.

User can De activate the Music Label by clicking on the Deactive icon in the Action column. However, if a Music Label is already used, then System won't allow the User to deactivate the record.

Once a record is deactivated, system will show Active button. If user clicks on Active button, record will get activated.

To Searh for any record, User can enter the Music Label name and then click on Search, it will show the result as per the search criteria. User can search name by single alphabet.

Click on page no, System will navigate the desired page, per page 10 records will be displayed.
