Music Theme

This module will be accessible only to the User who has rights for this module. The User who has rights for individual action (add/modify/activate/deactivate/view) can add, modify, view, activate, deactivate and change the Music Theme in the system.

This module is used to add a Music Theme in the system, which will be used later to assign the Music Theme.

Clicking on Music Theme -> Music Theme List, opens the Music Theme List page with all added Themes (as below).

User can also search for the Music Theme by clicking on the Search Button on the right side of the screen.

User can search through Theme in the Search box.

Add Music Theme:

To add a Music Theme, click on 'ADD' button on the Music Theme List page. This opens 'Add Music Theme' tab (like below) to add new Music Theme.

Add Music Theme will have the following fields:

Music Theme Name: The user can add the name of the Music Theme.

Status: This shows the status of the Theme, whether it is Active or Inactive.

Action: The user can 'Save' or 'Cancel' saving the Music Theme.

SAVE: If the user clicks on the 'Save' button, it will save the Music Theme and will show on the Music Theme List page.

CANCEL: Clicking on 'Cancel' will cancel the add operation and will go back to the Music Theme List page, without any changes.

Edit Music Theme:

The 'Edit' button is located on the Music Theme List page in the Action column, and is indicated by a Pencil icon. Clicking on the 'Edit' button, opens 'Edit Music Theme' tab (like below) to edit an existing Music Theme.

Edit Music Theme will have following fields:

Music Theme Name: The user can edit the existing name of the Music Theme.

Status: This shows the status of the existing Music Theme whether it is Active or Inactive.

Action: The user can 'Save' or 'Cancel' saving the Theme.

SAVE: If the user clicks on the 'Save' button, it will save the Music Theme and will show on the Music Theme List page.

CANCEL: Clicking on 'Cancel' will cancel the add operation and will go back to the Music Theme List page, without any changes.

Activate or Deactivate a Music Theme:

Activate Music Theme: 'Thumbs Up' icon activates that particular Music Theme.

Deactivate Music Theme: 'Thumbs Down' icon deactivates that particular Music Theme.
