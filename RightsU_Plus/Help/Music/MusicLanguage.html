Music Language

This module will be accessible only to the User who has rights for this module. The User who has rights for individual action (add/modify/activate/deactivate/view) can add, modify, view, activate, deactivate and change the Music Language in the system.

This module is used to add a Music Language in the system, which will be used later to assign the Music Language.

Clicking on Music Language>>Music Language List, opens the Music Language List page with all added Languages (as below).

User can also search for the Music Language by clicking on the Search Button on the right side of the screen.

User can search through Language in the Search box.

Add a Language:

To add a Music Language, click on 'ADD' button on the Music Language List page. This opens 'Add Language' tab (like below) to add new Language.

Add Language will have the following fields:

Language Name: The user can add the name of the Language.

Status: This shows the status of the Language, whether it is Active or Inactive.

Action: The user can 'Save' or 'Cancel' saving the Language.

SAVE: If the user clicks on the 'Save' button, it will save the Music Language and will show on the Music Language List page.

CANCEL: Clicking on 'Cancel' will cancel the add operation and will go back to the Music Language List page, without any changes.

Edit Music Language:

The 'Edit' button is located on the Music Language List page in the Action column, and is indicated by a Pencil icon. Clicking on the 'Edit' button, opens 'Edit Music Language' tab (like below) to edit an existing Music Language.

Edit Music Language will have following fields:

Language Name: The user can edit the existing name of the Language.

Status: This shows the status of the existing Language whether it is Active or Inactive.

Action: The user can 'Save' or 'Cancel' saving the Language.

SAVE: If the user clicks on the 'Save' button, it will save the Music Language and will show on the Music Language List page.

CANCEL: Clicking on 'Cancel' will cancel the Edit operation and will go back to the previous entry on the Music Language List page without any changes.

Activate or Deactivate a Music Language:

Activate Music Language: 'Thumbs Up' icon activates that particular Music Language

Deactivate Music Language: 'Thumbs Down' icon deactivates that particular Music Language.
