Revenue

Click on Revenue tab

To add Revenue, click on Add Button. A pop up page will appear as shown in the screenshot below

Currency Exchange Rate is displayed which cannot be edited

Select Title for which the given Revenue will be applicable

Royalty Recoupment Type can be selected from the drop down

Cost in Currency is auto populated

Enter the Fixed Cost in the Fixed Cost field

To Add Fixed Fees, click on the Fixed Fees tab in the Special Details section

Click on the Add icon to add the Fixed Fees

Select required cost type, comes from cost type master, in ‘Cost Type’ drop down. Enter amount in ‘Amount’ text box which should be less than or equal to ‘Fixed Cost’

Click on Save icon to save the Record

To Add Additional Expense, click on the Additional Expense tab in the Special Details section

Enter the Amount in the Amount field

Click on Save icon to save the Record

To Add Standard Returns, click on the Standard Returns tab in the Special Details section

Click on the Add icon to add the Standard Returns

Select the Party from the Party drop down column

User can select Commission Type either Cost or Theatrical Revenue or ROI

If user selects Cost then he/she need to select cost type from Cost Type drop down

If user selects Theatrical Revenue or ROI than only percentage can be added

Enter “Amount” which should be less than or equal to Fixed Cost

Enter “Percentage” which should be less than or equal to 100

Click on Save icon to save the Record

To Add Overflow, click on the Overflow tab in the Special Details section

It could be either Profit Sharing or Revenue Sharing or NA. By default it is Not Applicable

When Royalty Recoupment Type is 'No Recoupment' then Revenue Sharing will be available. If any Royalty Recoupment type is selected, then Profit Sharing will be available

Enter percentage for each Party and click save button to save the record. The Total Percentage should be equal to 100

Click on Edit button to modify the Details of Revenue or Click Delete to remove the record
