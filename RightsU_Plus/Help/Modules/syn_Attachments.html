Attachment

Click on Attachment Tab. It is used to attach any additional deal documents, if any.

New page appears with Add button at right side corner of the Page.

Adding Attachment

Click on Add button. A new row appears with empty Data.

Select Title against which user wants to provide attachment to.

Enter Attachment Title - Name of the file or short description.

Select Document Type - Data will be populated from Document Type master.

Browse Attachments: Choose file window appears. User can choose the necessary file for attachment up to 20 MB.

Click on Save button to save the record and Cancel to cancel saving.

Modifying the Attachment

Click Pencil Icon to modify the record or Click on Trash Icon to Remove the record.

Delete the Attachment

Click on Delete button for Remove the Attachment. An alert Message Appears as "Are You sure you want to delete this record" OK for Remove the Record and Cancel for back to the page without any operation.

Note: User can download the Attachment as on click of hyper link appears at Attachment column.
