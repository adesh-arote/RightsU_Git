Syndication - General

Following fields would be shown on top of the page which are consistent throughout the syndication module:

Agreement No. by default it will be "New Version #: 0001". It cannot be modified.

Deal Agreement Date, by default will show current system date. User can change the Deal Agreement Date. System will not allow selecting future date.

Deal Description field will be blank. User can enter the description here.

By default, Status will be Fully Executed.

Other section of page will be displayed with General, Rights, Reverse HB, Run Defn, Cost, Payment Terms, Attachments, Status History tabs.

Select Deal For, as either Movie or Program or Other. Other can be selected for Music.

Select Customer Type as either Content Owner or Platform operator. By default Content Owner will be selected.

Depending upon the Customer Type, User will be able to select Licensee for the deal.

Select Year Definition either FY (Apr-Mar) or Jan-Dec or Deal Year. By default, Deal Year will selected.

Business Unit will have multiple options: Hindi Movies, English Movies etc.

Select required Licensor Name(s) by clicking on respective name from the drop down.

Based on Licensee, Contact details will be displayed. On selection of contact, Phone and E-mail will be displayed.

Enter Reference No as per the Agreement.

Currency list box will show all active currencies from the master.

Select Category list box will show all the active categories from the Category Master

Sale Agent will be populated from Party Master. Contact, Phone, E-mail details will be populated as per selection.

Add Deal Movie

Click Add Movie button which shown in right side corner of the Deal Movie Grid.

A pop up with empty fields will be displayed with Save and Cancel button and search bar to Select movies.

Note: Acquisition titles must have sub-licensing, rights period not tentative and deal must be in approved state in order to populate titles in Add Movie search bar.

Select required title from the drop down and click on Save button. After saving, title will be added in grid with Title Language, Star Cast, Duration auto populated from Title's master. Also Movie type is as Premier by default but user can select as Library or NA and there is an option to Delete the title.

User can search with the title and apply paging size as required. Also on top, number of titles in deal will be shown in bracket.

There are 2 options in grid:

Search bar - User can search between added titles in this option.

Page Size - If there are multiple titles, user can put page size.

After choosing title and page size, click on Search to show updated data.

After filling all fields, page should look like this.

Click on SAVE button to save the information on General page and save deal and CANCEL to not save/update deal.
