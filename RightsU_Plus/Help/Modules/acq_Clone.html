Renew / Clone

To renew or clone an existing deal, user can use this option. This is helpful in creating a new deal when its related existing deal is renewed. It will be shown once the deal is approved.

When user clicks on Renew/Clone button, existing deal is copied and General tab is opened with an option to Replace the title.
User can replace the title with new one and click on Save Deal. This will copy rights of existing deal with new title.
