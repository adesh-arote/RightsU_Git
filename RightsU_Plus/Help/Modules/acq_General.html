General

Click Add New Deal Button to add new deal, a new Add Acquisition Deal page will appear.

Agreement No. by default it will be “New Version #: 0001”. It cannot be modified.

Deal Agreement Date, by default will show current system date. User can change the Deal Agreement Date. System will not allow selecting future date.

Deal Description field will be blank. User can enter the description here.

By default, Status will be Fully Executed.

Other section of page will be displayed with General, Rights, Reverse HB, Run Defn, Cost, Payment Terms, Attachments, Status History tabs.

Select Deal Type as Master Deal or Sub Deal. By default, Master Deal will be selected.

Select Mode of Acquisition as either Assignment or License or Own Production. By default Assignment will be selected.

Select Deal For, as either Movie or Program or Other. Other can be selected only for Sub Deal.

Licensee/Assignee will have multiple options: AXN Holdings, LLC, MSM Asia Ltd., etc.

Depending upon the Mode of Acquisition, User will be able to select either Assignor or Licensor or Producer/Line Producer.

Now select required licensor in Licensor/Assignor drop down. It will automatically populate the details of field "Phone" and "Primary Contact".

Master Deal to Link will be activated in the case of Sub Deal.

Select Year Definition either FY (Apr – Mar) or Jan – Dec or Deal Year. By default, Deal Year will selected.

Currency list box will show all active currencies from the master.

Business Unit will have multiple options: Hindi Movies, English Movies, Hindi Contect etc.

Select Category list box will show all the active categories from the Category Master

Enter Reference No as per the Agreement.

On selection of “Deal For” as “Movie” at the end of the page Deal movie list will shown as below.

Title drop down will display all the movies added to the deal. User can view only those title he wants by selecting from Title drop down.

User can set page size using Page Size field.

To add a movie in the deal Click Add Movie button, a pop up will be displayed as follows:

This drop down will show list of movies added in Title master.

User can add multiple movies. User will select all the required movie titles and click save.

On saving, Title Language, Star Cast, Movie Type and Duration will be auto filled as per selected movie.

Enter Notes, if required.

Select Movie Type either Premier or Library or N/A by selecting a radio button.

Enter Remarks up to 500 characters.

User will be allowed to edit the details of movie title until the reference of these titles in other tabs are removed.
