Syndication Deal

Syndication Deal is an agreement signed between a Licensor/Assignor and Licensee/Assignee which contains Rights syndicated or sold for a title.

This module will be accessible only to the user who has rights. To add, modify, view the data in/from the system, user must have rights for individual action.
