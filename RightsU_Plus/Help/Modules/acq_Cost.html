Cost

To add Cost details, click on 'Add' Button on top right corner of the page

Select Title for which the given Cost will be applicable

Cost type can be selected from the drop down

Enter the cost in 'Amount' field/p>

Select the Incentive as 'Yes' or 'No'

Enter Remarks upto 4000 characters

Click on the save icon located in the Action column to save the cost

Click on Cancel Button, the Record will not get updated

Click on Edit Button in the Action column to edit the added Record

Click on Delete Button in the Action column to delete the Record
