Run Definition

Run definition can be defined as number of exhibitions/runs syndicated for a title

To add run definition, in Run Definition tab click on ‘Add’ button

On clicking Add button, following page will be displayed.

Select required titles in Title. This Title field shows list of added titles in General tab.

For Platforms, Linear platforms added in Syndication Rights tab will populated and user can select platforms as required

Select No of Exhibition Runs as either limited or unlimited. By default limited will be selected.

A text box will be displayed to enter no of runs to play for the Title when limited is selected.

Note: - If user selects 'Unlimited' then no need to enter runs because user will get rights to play unlimited no of times within specified period.

Select Year Wise Run Definition either Yes or No. By default it will be selected as No

If Year Wise Run Definition is selected as Yes then the rights period will be split according to year definition selected in Rights tab. On the selection of Yes user will be given text boxes to enter no of runs according to year definition.

The sum of runs given over here should not cross runs given at ‘No of Exhibition’ otherwise system will show validation message.

Select Per Exhibition Day Rule Right (Yes/No).

If user selects “Yes” then select the applicable Rule from the drop down.

Day Start Time - On selection of Right Rule above, it populates as per the Data added in the masters.

Plays per Day - On selection of Right Rule above, it populates as per the Data added in the masters.

Note - Number of Runs should be greater than or equal to Plays per day.

Duration of Day - On selection of Right Rule above, it populates as per the Data added in the masters.

No. of Repeat - On selection of Right Rule above, it populates as per the Data added in the masters.

Repeat Within Exhibition Day - If 'Days' are selected, then select the particular Day. If 'Hrs' is selected, then enter the no of hours in the text box.

Click on SAVE to add the run definition and CANCEL to cancel saving run definition.
