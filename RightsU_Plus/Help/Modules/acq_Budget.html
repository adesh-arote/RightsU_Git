Budget

This tab allows to map Title with its related WBS Budget Code. This tab will populate titles only if Linear Rights are acquired for the same.

Clicking on Budget tab will open a window like this:

If Title name in deal matches with WBS Description, it will be auto mapped. Also user can manually map title with budget code by clicking on Add/Edit button. This will open a pop up as shown below:

Title - User can select title to map with Budget.

Search supports the following parameters. User can search with any of them. They are WBS Code, WBS Description, Studio/Vendor, Original/Dubbed, Short ID, Status, Sport Type.

Link Title to Budget will link title with the budget.

To delete the budget and title mapping, click on Trash Icon. It will remove the mapping of budget and title.
