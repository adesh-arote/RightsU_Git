Approval List

Approval List Tab show list of Acquisition, Syndication as well as both deals which are sent for approval i.e. waiting for approval. This list will only be available to authorized user who has the rights to Approve/Reject Deals, as per their respective business unit

When user opens the Approval List, which will open a pop-out window to review the Acquisition, Syndication as well as Both deals which has been sent for approval.

Approving/ Rejecting Acquisition Deal:

Select any Acquisition Deal, which will display Workflow Status as waiting for authorization from Legal reviewer.

Approve/Reject page should be open with all fields in view mode and at the end of the page APPROVE, REJECT and BACK TO LIST button will be displayed.

Approval/ Rejection Remark: Remark field will be blank. User has to enter the remark here.

The remark field is manditory.

Click Approve button.

A confirmation message will appear as "Are you sure, you want to Approve this deal?" with OK and CANCEL buttons. Click OK and an alert will be displayed as deal approved. Approval status should be "Approved".

Similarly, Click Reject button.

A confirmation message will appear as "Are you sure, you want to Reject?" with OK and CANCEL buttons. Click OK and an alert will be displayed as deal rejected.

Approving/ Rejecting Syndication Deal:

Select any syndication Deal, which will display Workflow Status as waiting for authorization from Legal Reviewer.

Approve/Reject page will open in non-editable mode and at the end of the page Approve, Reject and Back to List button will be shown.

Approval/ Rejection Remark: Remark field will be blank. User has to enter the remark here.

The remark field is manditory.

Click Approve button. A confirmation message will appear as "Are you sure, you want to approve this deal?" with OK and Cancel buttons. Click OK to approve deal and Cancel button to cancel approval.

Approve/Decline Syndication Deal page gets closed and Syndication Deal list page shown, and Sent for approval Syndication Deal Approval status is "Waiting for Authorization 'Role Name'" shown in case multi-user approval role is assigned or else it shows "Approved".

"Waiting for Authorization 'Role Name'" The Role Name shows as per Approval work flow assigned for e.g. Syndication Deal Approval Syndication Deal assigned for 2 stages 1. System Admin 2.Account Manager. Than is should shown as "Waiting for Authorization System Admin".

Log in with System Admin User for same (Approval 1) process will follow for Approval 2 or click Cancel Syndication Deal should not be approved and main page should be shown.

Reject Syndication Deal.

Enter reason for rejecting deal.

Click Reject button. A confirmation message will appear as "Are you sure, you want to Reject this deal?" with OK and Cancel buttons. Click OK to reject the deal and Cancel button to cancel rejection.

Approve/Decline Syndication Deal page should be closed and Syndication list page should be shown, and Sent for approval Deal Approval status should be Rejected. System will show view, Edit, Re-Send for Approval Button.

Note: Rejected deal can be edited and send for approval by approver himself.

In case any user will decline the deal than system should follow the same process once again, send for approve, Approval 1, Approval 2.
